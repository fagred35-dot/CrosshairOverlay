# CODEX Crosshair Overlay — v2.1.4

## Исправления
- **Окно назначения хоткеев теперь открывается и принимает клавиши.** Убран флаг `WS_EX_NOACTIVATE`, добавлен `PauseTopmost` при показе диалога.
- Тема UI рендерится **градиентными плашками** (Dark / Light / OLED / Neon) вместо обрезанного текста.
- Обычный автокликер ограничен **60 CPS** (как и просили — не грузит ПК).

## Новое
- **Burst теперь добавляет клики поверх обычного потока.** При нажатии ЛКМ дополнительно отправляется N пакетных кликов через `SendInput` одним вызовом — высокая пиковая скорость без нагрузки на CPU. Лимит Burst увеличен до 200.
- **Диалог «Достижения» в стиле приложения:** градиентный заголовок, карточки с иконкой, прогресс-барами и цветовыми акцентами. 9 ачивок с прогрессом. Новый файл `AchievementsForm.cs`.

Закрывает замечания после v2.1.3: «хоткеи не открываются», «тема UI выглядит плохо», «обычный автокликер до 60 CPS + Burst добавляет доп. клики».
