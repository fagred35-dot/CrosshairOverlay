# CODEX Crosshair Overlay — v2.1.3

## ⚠ Критический фикс
- **exe снова запускается.** В v2.1.2 single-file publish собирался без `IncludeAllContentForSelfExtract`, из-за чего при запуске Windows писала «The application to execute does not exist: 'CrosshairOverlay.dll'». Теперь все managed и native зависимости встроены в один файл.

## 🎨 Новый UI
- **Табы:** настройки разделены на 6 категорий — **Вид / Эффекты / Клик / Хоткеи / Дополн. / О прог.**
- **Ctrl+F** — поиск по всем настройкам.
- **Колёсико мыши над слайдером** меняет значение.
- **Правый клик по слайдеру** = сброс к дефолту.
- **Темы UI:** Dark / Light / OLED / Neon.

## 🎯 4 новых стиля прицела
- **DoubleCircle** — двойное кольцо.
- **DashedCross** — пунктирный крест.
- **TriangleUp** — треугольник вверх.
- **SerifCross** — крест с засечками.

## ⌨ 4 новых хоткея
- **Ctrl+Shift+O** — открыть папку со скриншотами.
- **Ctrl+Shift+]** — следующий пресет.
- **Ctrl+Shift+[** — предыдущий пресет.
- **Ctrl+Shift+K** — вкл/выкл Анти-AFK.

## 🖱 Автокликер
- **Анти-AFK:** клик каждые N секунд при отсутствии активности.
- **Лимит кликов за сессию** + авто-стоп.
- **Таймер работы** (авто-стоп через N минут).
- **Jitter Aim:** случайные микро-смещения курсора.

## 💾 Профили
- **Импорт/экспорт профиля** в JSON-файл.
- **Share-код прицела** `CX1:...` через буфер обмена.

## 🖥 Система
- **Автозапуск с Windows** (HKCU\\Software\\Microsoft\\Windows\\CurrentVersion\\Run).
- **Portable-режим:** положите `portable.flag` рядом с exe — настройки будут в той же папке.
- **Скрытие прицела в fullscreen-играх** (опция).
- **Детектор конфликтов хоткеев** — предупреждает о дубликатах.

## 📊 Статистика
- **Время использования** за день и за всё время.
- **Daily streak.**
- **Достижения:** 1M кликов, 10 часов, 500+ CPS, 7-day streak.

## 🔧 Под капотом
- Новый файл `FeaturesExtra.cs` с утилитами:
  `AutostartManager`, `PortableMode`, `ProfileIO`, `UsageTracker`, `HexColorForm`, `UiThemePresets`, `ShellHelper`, `HotkeyConflictDetector`, `FullscreenDetector`.
- `CrosshairStyle` enum расширен 4 значениями.
- `HOTKEY_COUNT` увеличен с 17 до 21.
- `SettingsData` расширен 9 полями (сохранение/загрузка обновлены).
- Версия сборки: **2.1.3.0**.
