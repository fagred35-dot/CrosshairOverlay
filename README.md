# CODEX Crosshair Overlay

**Полноэкранный оверлей прицела + автокликер для Windows**

Легковесная утилита без установки — один `.exe` файл. Рисует кастомный прицел поверх любой игры/приложения и включает встроенный автокликер с многопоточной архитектурой.

---

## Возможности

### Прицел
- **15 стилей прицела:** Cross, Circle, Dot, CrossWithCircle, Chevron, T-Shape, Diamond, Arrow, Plus, X-Shape, Triangle, Crosshairs (снайперский), Square Brackets, Wings, Custom Image
- **Своя картинка** — загрузи PNG/JPG и используй как прицел
- **Ручной поворот** (0–360°) и автовращение с настройкой скорости
- **Цвет, градиент, радужный режим**
- **Контур и тень** с настройкой толщины и цвета
- **Центральная точка** с пульсацией
- **Регулировка:** размер, толщина, отступ, прозрачность, смещение X/Y

### Автокликер
- **От 5 до 1000 кликов/сек** (CPS) с честным масштабированием
- **Многопоточная архитектура** — автоматически распределяет клики по потокам
- **Режимы:** зажатие ЛКМ / постоянный клик
- **ПКМ режим** — кликает правой кнопкой
- **Рандомная задержка** — антидетект вариация задержки
- **Аварийная остановка** — клавиша `Escape` мгновенно останавливает кликер

### Динамический прицел
- Раздвигается при движении мыши (как в CS)
- Настраиваемая чувствительность и максимальный разброс
- Скорость возврата

### Эффекты
- Хитмаркер при клике
- Пульсация при выстреле
- Сглаживание (Anti-Alias)

### Интерфейс
- **Кастомное меню настроек** с градиентными темами
- **Русский / English** — автоопределение языка
- **Авто-обновление** через GitHub Releases
- **Трей-иконка** с быстрым доступом

---

## Хоткеи

| Клавиша | Действие |
|---------|----------|
| `Insert` | Показать / скрыть прицел |
| `Ctrl+Shift+A` | Вкл/выкл автокликер |
| `Ctrl+Shift+S` | Открыть настройки |
| `Ctrl+Shift+C` | Сменить стиль прицела |
| `Escape` | Аварийная остановка автокликера |

## Установка

1. Скачай `CrosshairOverlay.exe` из [Releases](https://github.com/fagred35-dot/CrosshairOverlay/releases)
2. Запусти — готово. Установка не нужна

**Требования:** Windows 10/11, x64

## Сборка из исходников

```bash
dotnet publish -c Release -r win-x64 --self-contained -p:PublishSingleFile=true
```

Выходной файл: `bin/Release/net6.0-windows/win-x64/publish/CrosshairOverlay.exe`

## Технологии

- C# / .NET 6.0 / WinForms
- GDI+ рендеринг с `UpdateLayeredWindow`
- Win32 API: `mouse_event`, `RegisterHotKey`, `timeBeginPeriod`
- Многопоточный кликер с `Stopwatch`-throttling
