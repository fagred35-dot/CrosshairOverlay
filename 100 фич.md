# 100 идей для CODEX Crosshair Overlay

> Ранжировано по категориям. Пометки: **[E]** — легко, **[M]** — средне, **[H]** — сложно/рискованно.

## Прицел — стили и рендер (1–15)
1. **[E]** Двойное кольцо (circle внутри circle, настраиваемый зазор).
2. **[E]** Пунктирный прицел (DashPattern у Pen).
3. **[E]** Стиль «скобки ASCII» — `< >`, `[ ]`, `{ }`.
4. **[M]** SVG-загрузчик для картинок-прицелов (через SkiaSharp или ручной парсер путей).
5. **[M]** Анимация пульсации всего прицела (scale sine wave), не только точки.
6. **[M]** «Breathing» режим — медленное изменение opacity 50→100%.
7. **[E]** Треугольник вверх как отдельный стиль (сейчас только triangle down).
8. **[E]** Крест с засечками на концах (serif-cross).
9. **[M]** Радиальный градиент от центра наружу.
10. **[M]** Конический градиент (как в CSS `conic-gradient`).
11. **[M]** Режим «negative» (инвертирование цветов под прицелом через `UpdateLayeredWindow` с другим blend).
12. **[E]** Прицел-«таймер»: вращающийся сектор как индикатор зарядки.
13. **[H]** Прицел, рендерящийся поверх курсора, а не по центру экрана (low-level hook для позиции).
14. **[M]** Стиль «reticle» — как в снайперских прицелах, с делениями и метками дальности.
15. **[M]** Анимация «появления» прицела при показе (fade + scale).

## Динамика и реактивность (16–25)
16. **[M]** Изменение цвета при попадании (интеграция с цветом пикселя под прицелом).
17. **[H]** Определение красного «хит-маркера» игры и триггер нашего эффекта.
18. **[M]** Расширение при стрельбе (как recoil spread), конфигурируемая длительность/амплитуда.
19. **[E]** Цвет прицела от состояния клавиши Shift (например, прицеливание).
20. **[M]** Переключение стиля/цвета по активному окну (профили на EXE игры).
21. **[M]** Авто-скрытие прицела в меню (детект по заголовку или по скриншоту центральной области).
22. **[E]** Индикатор «опущенного» хоткея — тонкий цветной ободок.
23. **[M]** Реакция на FPS: при низком FPS тускнеет.
24. **[H]** Интеграция с Wooting/Razer/SteelSeries SDK для подсветки, синхронной с прицелом.
25. **[M]** «Heat» визуализация: чем чаще стреляешь, тем ярче/горячее цвет.

## Профили и пресеты (26–35)
26. **[E]** Импорт/экспорт всего профиля в JSON-файл.
27. **[E]** Импорт/экспорт одного прицела в строку (clipboard code).
28. **[M]** CS2 crosshair code импорт — распарсить и сконвертировать.
29. **[M]** Быстрый свитч профилей хоткеями 1–9.
30. **[M]** Авто-профиль по активной игре (switch при смене foreground window).
31. **[E]** «Последний использованный» — быстро вернуться к предыдущему.
32. **[M]** Случайный прицел каждые N минут (для тренировки адаптации).
33. **[E]** Облако пресетов (публичный репо JSON на GitHub, загрузка в «Сообщество»).
34. **[M]** Лайки/рейтинг пресетов (локально, сохраняет топ).
35. **[M]** A/B-тест: два прицела сменяются по хоткею для сравнения.

## Автокликер (36–50)
36. **[M]** Кастомный паттерн интервалов (например, 80-20-80 мс — «очередь»).
37. **[M]** «Анти-AFK»: клики раз в N секунд со случайным джиттером.
38. **[M]** Клик по конкретной координате (не текущая позиция курсора).
39. **[M]** Кликер-мульти-точка: круг позиций, обходит по очереди.
40. **[E]** Лимит общего количества кликов за сессию (auto-stop).
41. **[E]** Timer-таймер работы (стоп через N минут).
42. **[M]** Двойной/тройной клик как одно событие (с настраиваемой паузой).
43. **[M]** SMB: средняя кнопка мыши режим.
44. **[M]** Клик по X/Y из CSV-скрипта (макрос).
45. **[H]** Запись макроса мыши и воспроизведение (как TinyTask).

48. **[M]** Toggle vs Hold режим на отдельном хоткее.
49. **[M]** Игнор кликов в заданных окнах (по названию процесса).
50. **[H]** Distinguish LMB/RMB источника — разные профили автокликера.
